# Informações gerais dos países, territórios e nações não reconhecidas pela ONU.

Informação detalhada de todos os países, territórios e territórios não reconhecidos pela ONU. O sistema apresenta um card com algumas informações e, ao clicar nesse card, abre um modal com várias informações sobre esse país, tais como moeda, idiomas falados, fronteiras, etc. 

## API Utilizada

https://restcountries.com
